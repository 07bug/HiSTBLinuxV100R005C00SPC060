==================
Usage descriptions
==================
1 sample_cipher.c:
  Show how to use the interfaces for normal encryption and decryption.
2 sample_multicipher.c:
  Show how to use the interfaces for multiple blocks encryption and decryption.
3 sample_cbcmac.c:
  Show how to use the interfaces for calculating the message authentication code using AES-CBC.
4 sample_hash.c:
  Show how to use the interfaces for calclating message of hash.
5 sample_anticopy.c:
  Show how to use the interfaces for encryption and decryption of copy protection.
6 sample_rng.c:
  Show how to use the interfaces for generating random values.
