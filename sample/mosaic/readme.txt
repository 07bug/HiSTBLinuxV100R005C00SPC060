Purpose: to demonstrate the playing in mosaic mode. Four windows are supported if there is sufficient memory space.
      dvb_mosaic: TSs come from cables. 
      ip_mosaic: TSs come from the IP network.
Command: dvb_mosaic freq srate 
      freq: frequency  
      srate: symbol rate 
      ip_mosaic MultiAddr UdpPort
      MultAddr: multicast address  
      UdpPort: multicast port
